# Portfolio Site

Web制作学習の成果をまとめたポートフォリオサイトです。

レイアウト設計からコーディング、レスポンシブ対応まで一貫して制作しました。

## 使用技術

- HTML5
- CSS3
- JavaScript
- jQuery
- Git / GitHub

## 実装内容

- レスポンシブ対応
- ハンバーガーメニュー実装
- GitHub Pagesによる公開
- Flexboxを用いたレイアウト構築
- ポートフォリオ作品掲載
- お問い合わせ導線の実装

## 工夫した点

- シンプルで見やすいデザインを意識
- スマートフォン表示を考慮した設計
- 配色や余白を調整し視認性を向上
- 作品をカテゴリーごとに整理

## 制作期間

約2週間
（職業訓練と並行して制作）

## 感想

ポートフォリオサイト全体の構成を考えながら制作することで、情報設計やレイアウト設計の重要性を学びました。

制作中に発生した課題は開発者ツールやGitHub、AIを活用しながら解決し、完成まで継続して取り組みました。
